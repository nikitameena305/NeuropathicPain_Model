# Project 2: DRG Hyperexcitability + SDH Disinhibition Combined Model

## Main question
Is pain output strongest when both peripheral DRG excitability and central SDH disinhibition occur together?

## Biological idea
Neuropathic pain is usually not caused by only one thing.

It can involve:

1. Peripheral sensitization:
   DRG neurons become hyperexcitable and send more pain input.

2. Central disinhibition:
   SDH inhibitory GABA/glycine brake becomes weaker.

Together:

DRG hyperexcitability = more accelerator
SDH disinhibition = weaker brake

So projection neurons can fire more strongly.

## Conditions

| Condition | DRG state | SDH GABA state |
|---|---|---|
| C1 | Normal DRG | Normal GABA |
| C2 | Hyperexcitable DRG | Normal GABA |
| C3 | Normal DRG | Reduced GABA |
| C4 | Hyperexcitable DRG | Reduced GABA |

## Expected result
C4 should show the strongest SDH projection output.

## Connection to Project 1
Project 1 showed that reducing GABA increases SDH output.
Project 2 adds DRG hyperexcitability on top of that.

## Connection to Tigerholm et al. 2014
Tigerholm's C-nociceptor model shows that axonal excitability and sodium dynamics influence nociceptor firing and conduction behavior.
In this project, DRG hyperexcitability represents stronger peripheral nociceptive drive entering the SDH.
